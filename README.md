# AI 工资预测系统（全栈项目）

基于 Python + FastAPI + 机器学习 + MySQL + 前端 的企业级 AI 预测平台

## 技术栈

- 后端：FastAPI / 异步 / 鉴权 / 日志
- 数据库：MySQL
- 机器学习：线性回归（工资预测）
- 前端：Dashboard 可视化大屏
- 部署：支持本地 / 服务器上线

## 核心功能

- 员工数据管理
- 部门多维度统计
- AI 工资智能预测
- 前后端分离架构
- 企业级 API 服务

## 项目亮点

- 纯 Python 全栈开发
- 模型训练 + API 部署一体化
- 前端可视化大屏展示
- 可直接上线部署

## 运行方式

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
